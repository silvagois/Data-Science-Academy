Tipos de apendizagem

Aprendizagem Supervisionada: 
É quando entregamos ao algoritimo um conjunto de dados de entrada e de saida, assim o algoritimo tenta encontrar uma correlação matemática destes dados.
E o termo usado sempre que o programa é treinado sobre um conjunto de dados pre definido
Uso de dados historicos

Classificação: Tem como algo variáveis qualitativas ou categóricas, exemplo validar uma imagem e classifica-la como homem ou mulher

Regressão: quanto vai custar determinado produto ou valor de uma ação

Aprendizagem Não Supervisionado:
Entrego somente os dados de entrada e não entrego os de saida, esperando que o algoritmo descubras quais serão as saidas baseado nos padroes destes dados. Exemplo segmentação de clientes, tenho os dados dos clientes e assim o algoritimo irá agrupar por segmentação os dados 


Aprendizagem Semi Supervisionada
Aprendizagem por Reforço, mais para games

Deep Learning:
Voltado para AI, um tipo de aprendizado de máquina, redes neurais, Deep Neuro Networks

O que é um modelo preditivo?

ETAPAS DO MODELO PREDITIVO
Coleta de Dados, 
Exploração e Preparação(Analise Exploratória dos Dados:limpeza dos dados, eliminação dos dados desnecessários, recodificação dos dados, Transformação, Organização, Tratamento de dados faltantes etc...), 
Treinamento do Modelo, 
Avaliação do Modelo, 
Otimização do Modelo

Acurácia: taxa de acerto do modelo > 90%
Deploy em produção: Gerar um produto ou serviço atraves deste modelo
